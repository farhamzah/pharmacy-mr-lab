# Stage 3 Known Issues

- Procedural model belum setara GLB realistis.
- Label sprite belum memakai per-frame camera-facing constraint eksplisit.
- Audio bisa diblokir browser sampai ada user gesture.
- Haptic feedback bergantung pada gamepad actuator yang diekspos browser.
- Object scale calibration hanya berlaku untuk objek yang dibuat setelah pilihan berubah.
- Spatial anchor permanen belum tersedia.
- Serbuk dan cairan belum memakai physics nyata.
- Material dynamic pada visual mixing masih bisa dioptimasi lebih lanjut dengan pooling.
