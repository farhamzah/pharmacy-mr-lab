# Stage 2 Known Issues

- Spatial anchor permanen belum digunakan; posisi meja masih fallback relatif session.
- DOM overlay dapat tidak konsisten di beberapa versi Meta Quest Browser.
- Placeholder objek belum memakai skala dan bentuk alat farmasi realistis.
- Massa sample dan homogenitas memakai random terkontrol, bukan simulasi fisika.
- Progress step masih linear sederhana dan belum mendukung branching remediation.
- Result history hanya localStorage lokal browser, belum ada sinkronisasi atau export.
- Desktop Module Demo membantu testing logic, tetapi tidak menggantikan validasi passthrough AR di headset.
