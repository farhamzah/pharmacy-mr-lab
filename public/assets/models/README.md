# Local GLB Asset Folder

Place only legal, local GLB/GLTF assets here. Stage 5 works without GLB files because procedural fallbacks are used automatically.

Recommendations:
- Use `.glb` where possible.
- Keep models low-poly for Meta Quest Browser.
- Prefer textures 1024px or smaller.
- Match filenames from `src/assets/AssetManifest.ts`, or update the manifest.
- Enable `Use External 3D Assets` in the Home UI to test local assets.
